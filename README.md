# Florr_Bot
Implement Florr game AFK automation. Currently not doing very well, welcome experts to improve.

目前写的不太好，欢迎大佬指正

## 实现功能
### 一、网页监测与自动化任务执行
- 实时监测浏览器标签页访问 https://www.florr.io
- 自动识别游戏画布并执行自动化操作
- 持续挂机功能，维持游戏活动状态
### 二、图像识别与游戏操作自动化
- 通过 Canvas API 获取游戏画面像素数据
- 基于颜色匹配检测怪物、掉落物、玩家位置
- 精准鼠标控制：移动、点击、拖拽操作
- 自动化策略：优先攻击最近的怪物，拾取范围内的掉落物
### 三、游戏机制说明
- 弹出界面提供游戏概念解释（挂机、刷怪、掉落物、生命值）
- 提供 florr.io 访问链接
### 四、防挂机检测机制
- 监测无操作超时状态
- 自动检测挂机验证界面
- 模拟人类行为完成拖拽验证
## 安装步骤
1. 打开 Microsoft Edge，进入 edge://extensions/
2. 启用"开发者模式"
3. 点击"加载压缩的扩展程序"
4. 选择 ……（这里是你的下载目录）\edge-extension 目录
## 使用方法
1. 安装扩展后，打开 https://www.florr.io 游戏页面
2. 点击扩展图标打开控制面板
3. 启用自动化功能并调整参数
4. 点击"启动"按钮开始自动挂机
扩展界面提供实时状态显示（日志、生命值、怪物数量、击杀数等）和多项可调设置。
